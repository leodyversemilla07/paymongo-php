Release notes for v1.1.0

This release aligns the SDK more closely with the current PayMongo documentation and fixes several production-impacting transport issues. It adds Payment Intent attach support, Payment Method update support, Link retrieval by reference number, and Bearer-token support for `customersV2`. It also corrects stale merchant capability, installment plan, onboarding, and document-upload routes, improves multipart/raw-body handling, and hardens webhook signature parsing.
